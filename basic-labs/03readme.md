creating a simple dhcp server using:

- 3 pcs
- 1 switch
- 1 server

open the server and manually assing itself an IP

from the services tab in server, select DHCP, turn it on and do the following

there is a prexisiting scope, add the starting value of the IP in it, make sure it is from the same scope which you have assigned to your dhcp server manually, save it

from all the PCs, select the DHCP to be assigned dynamically
